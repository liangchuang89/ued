      主线命令，控制总的PHP运行流程

      构造函数配置的作用

      array(
	'get' => array('log'=>1),
	//配置插件,插件只能有一个触发的参数，而且不要调用一样的参数
	'plugins' => array(
		'codeviewer' => array(
         'trigger' => 'viewsource',
         'use' => array('page','component'),
         'description' => '查看源码插件启动所需触发的$_GET参数'
     )
	 )
)
      get会在P::initGetPatm时使用，用来替换$_GET参数。
      plugins配置插件启动项：
      • trigger:插件触发的参数
      • use:插件都使用了那些参数
      • description:插件作用描述
